## Global Setup & Requirements

Python Version: 3.10+ (3.11 recommended)
Dependencies: pygame, tkinter, PyQt5, matplotlib, and others per project (requirements.txt)

##Setup:

git clone https://github.com/<your-username>/fun-python-projects.git
cd fun-python-projects

# Optional: Create a virtual environment
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt


Run any project from its folder as shown above.
